# 🔋 EnergyLens-AI — Autonomous Energy Optimization Platform

An AI-powered energy analytics system using London Smart Meter data to forecast consumption, identify usage patterns, detect anomalies, and deliver optimization insights for smarter energy management.

## Dataset
**Source:** [Smart Meters in London — Kaggle](https://www.kaggle.com/datasets/jeanmidev/smart-meters-in-london)

Download and extract all files into `data/raw/`.

## Project Structure
```
EnergyLens-AI/
├── notebooks/           # Experimentation notebooks (run in order)
├── data/
│   ├── raw/             # ← Extract Kaggle dataset here
│   └── processed/       # Cleaned data (generated by notebooks)
├── models/              # Saved trained models
├── src/                 # Reusable modules (Phase 2)
├── app.py               # Streamlit dashboard (Phase 2)
└── requirements.txt
```

## Setup
```bash
pip install -r requirements.txt
```
